# drcr
## Program for Estimating Dark Figures of Crime
Econometric method to estimate the dark figure and dark rate of illegal activity. This method is described in detail in the paper "Estimating Dark Figures of Crime" by Bellert, Guenster and Kozbur (2025).

## Requirements
Install R: https://cran.rstudio.com/

Install RStudio: https://posit.co/downloads/

In your RStudio console:

`library(devtools)`

`install_github("belln1/drcr");`

`install.packages(c("DescTools"))`


## Getting Started
Type `?drcr` and run the described examples.

Run R file for a basic example:  
examples/test_dark_rate.R

Run R Markdown with code examples for simulation, summary statistics and estimates:  
examples/examples.Rmd
